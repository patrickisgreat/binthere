# binthere
**an app to prevent me from saying: "where the fuck is this?"**


**Requirements:**

1.) be able to scan a qr code on a bin and know what is in that bin.  
2.) be able to store pictures of items and associate them to records of items.
3.) Ability for Ai to analyze the image and pre fill out the data to the best of its ability then giving the user the chance to verify / edit.
        as in user takes photo of contents of bin or drawer, ai analyzes image, says this is the stuff that's in it and then can populate our db
4.) be able to check an item out of a bin.  
5.) be able to check an item back into a bin.  
6.) Bins should also have meta data like what zone they are in.  
7.) reminders to check items back in.  
8.) ability to send notifications to people who have something checked out.  
9.) Ability to remove items from the system if sold or damaged.  
10.) Ability to change an items bin  
11.) ability to store details about an item using custom fields and tags.
